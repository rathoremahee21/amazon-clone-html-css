# Amazon Clone

A front-end Amazon homepage clone built using HTML and CSS.

## Features

* Amazon-inspired navigation bar
* Search bar UI
* Delivery location section
* Hero banner section
* Product showcase cards
* Flexbox-based layout
* Hover effects and styling

## Technologies Used

* HTML5
* CSS3
* Font Awesome
